# My Marketplace

Platform terpercaya untuk jual beli dan sewa barang dengan desain modern dan responsif.

## 🎯 Tentang Project

My Marketplace adalah halaman arahan profesional untuk bisnis jual beli dan sewa barang. Dibangun dengan React 19, Tailwind CSS 4, dan shadcn/ui.

## ✨ Fitur Utama

- Hero section yang menarik
- Tiga kategori layanan (Beli, Jual, Sewa)
- Showcase produk unggulan
- Call-to-action yang jelas
- Fully responsive design
- Modern Minimalist design

## 🎨 Design System

- **Primary Color**: Navy Blue (#1F2937)
- **Accent Color**: Vibrant Teal (#14B8A6)
- **Typography**: Sora (Headlines) + Inter (Body)

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
